- 00:00 <https://youtu.be/GKHA6OUeywo?t=0m0s> 導入：メモ作成・管理の悩みを解決する「コパイロット」
    - Obsidianでのメモ作成時における「どこに何を書けばいいかわからない」「表現が思いつかない」といった悩みを解決するプラグイン。
    - AIが「副操縦士（コパイロット）」として、ノート作成を強力にアシストしてくれる。
- 01:43 <https://youtu.be/GKHA6OUeywo?t=1m43s> Copilotプラグインの概要と特徴
    - ObsidianのAI系プラグインの中でダウンロード数No.1（130万ダウンロード以上）を誇る、最も人気のあるプラグイン。
    - PDFを読み込ませて壁打ち、Web検索、Obsidian内のメモ検索や関連ノートの抽出など、多様な操作をサイドバーから行える。
    - 基本機能は無料で利用可能。OpenAIやGoogle（Gemini）など、様々なAIモデルを連携して使用できる。
- 02:51 <https://youtu.be/GKHA6OUeywo?t=2m51s> Copilotプラグインのインストール手順
    - 1. Obsidianの左下「歯車マーク（設定）」→「コミュニティプラグイン」を開く。
    - 2. 「閲覧」ボタンから「Copilot」と検索し、130万DL以上の公式プラグインをインストールして有効化する。
    - 3. 完了すると、左サイドバーにチャットアイコンが表示され、右側にチャット画面が展開する。
- 03:31 <https://youtu.be/GKHA6OUeywo?t=3m31s> APIキーの設定（Google AI Studioの活用）
    - 初期状態のままチャットを送るとエラーが出るため、AIを動かすための「APIキー」の設定が必要。
    - 1. ブラウザで「Google AI Studio」にログインし、左下の「Get API key」からAPIキーを生成・コピーする。
    - 2. Obsidianに戻り、Copilotプラグインの設定画面（General API Keys）の「Gemini」の項目にコピーしたキーをセットする。
    - 3. 使用するAIモデルを選択（動画内では「Gemini 3.1 Flash Light」等を選択）することで、チャットが正常に動くようになる。
- 05:23 <https://youtu.be/GKHA6OUeywo?t=5m23s> 実践：Copilotプラグインを使ったメモ作成と壁打ち
    - チャット欄で「@（アットマーク）」を入力すると、コンテキスト（フォルダやアクティブノート）を指定できる。
    - 開いているノート（Active Note）を指定し、「この内容を補足して」と指示を出すだけで、AIがノートを読み取って自動で追記テキストを生成してくれる。
    - 右サイドバーでAIと壁打ちしながら思考を深め、生成された内容をノートに反映（コピペ）させていく運用が可能。
- 06:15 <https://youtu.be/GKHA6OUeywo?t=6m15s> Copilotの強み（他のAI連携との違い）
    - 単にChatGPTの画面をObsidian内に呼び出しているだけの場合（レベル1）は、Obsidian内のファイルやフォルダを直接指定して読み込ませることができない。
    - Copilot（レベル2）であれば、「@」を用いてObsidianの保管庫（Vault）の中身を認識させながら操作できる点が最大の魅力。
    - さらに高度な複数ファイルの読み込みや画像生成、複雑なタスク自動化（AIエージェント化）を求める場合は、今後紹介される「Cloudian（クラウディアン）」というプラグインもおすすめ。
- 08:25 <https://youtu.be/GKHA6OUeywo?t=8m25s> エンディング＆動画の裏話
    - 今回は無料モデル（Geminiの軽量モデル）を使用したため、やや出力の精度や速度に物足りなさ（イライラ）を感じる場面もあった。
    - 初心者の最初のAIプラグインとしては導入しやすく最適だが、より賢く動かしたい場合は有料の優秀なモデルを設定するか、AIエージェント特化の「Cloudian」を検討するのが良い。
