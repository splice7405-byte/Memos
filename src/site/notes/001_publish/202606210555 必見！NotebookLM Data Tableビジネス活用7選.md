- 00:00 <https://youtu.be/N77DlHnbD7Y?t=0m0s> オープニングと概要
    - 2025年12月に追加されたNotebookLMの「データテーブル（Data Table）」機能のビジネス活用術を解説。
    - 実践的な7つのビジネスシーンを紹介し、日常業務への応用方法を提案する。
- 01:44 <https://youtu.be/N77DlHnbD7Y?t=1m44s> データテーブル機能とは何か
    - ユーザーの指示に従い、ソース（登録された情報）の内容を表形式（構造化データ）に素早く整理する機能。
    - シンプルな情報であれば30秒以内、複雑な情報でも1分程度で生成が完了する。
    - 最大のメリットは、Googleスプレッドシートへボタン一つでエクスポートでき、その後の編集・共有・加工が自由に行える点。
- 02:40 <https://youtu.be/N77DlHnbD7Y?t=2m40s> ビジネスにおけるデータテーブルの重要性
    - スライドやインフォグラフィックは他人に「伝える」のには有効だが、生成に時間がかかりデータ加工もできない。
    - 一方、データテーブルは複雑なデータを一目で比較・俯瞰でき、営業会議やミーティング、顧客対応などの業務効率を劇的に向上させる。
- 04:31 <https://youtu.be/N77DlHnbD7Y?t=4m31s> シーン1：契約書の比較とリスクチェック
    - 2つの異なる契約書を読み込ませ、目的、業務内容、報酬体系、秘密保持などの異なる点を表形式で一瞬で抽出。
    - 「自社（乙）にとって不利な条文」を自動チェックさせ、修正の方向性まで整理可能。弁護士への相談時間を大幅に短縮できる。
- 06:58 <https://youtu.be/N77DlHnbD7Y?t=6m58s> シーン2：履歴書のポイント整理と採用先行
    - 複数人（デモでは6名分）の履歴書から、職歴、資格、志望動機、自己PRを一覧表に整理。
    - 自社の採用評価基準をソースに含めることで、評価基準に沿った「通過／落選」の判定と、その選考理由を自動で分析・出力させることが可能。
- 08:48 <https://youtu.be/N77DlHnbD7Y?t=8m48s> シーン3：競合他社のサービス比較
    - Web検索などで得た競合コンサル会社の情報から、ターゲット顧客、提供サービス領域、支援スタイル、従業員数などをマトリクス化。
    - 自社の強みをどこに伸ばしていくかの戦略立案・整理が容易になる。資料がない場合はWebサイトのURLを入力するだけでも対応可能。
- 10:27 <https://youtu.be/N77DlHnbD7Y?t=10m27s> シーン4：導入サービスの選定と役員会決済
    - 経費精算Saasなどの複数サービスURLを読み込ませ、「社員数50名規模」といった自社の前提条件を元に優先度と理由を比較整理。
    - 自社の現状と課題を掛け合わせることで、役員会議での決済用として、メリットや選定理由を順位付けした説得力のある比較表が作れる。
- 13:07 <https://youtu.be/N77DlHnbD7Y?t=13m7s> シーン5：営業会議のアクションプラン（To Do）作成
    - 会議で挙がった課題を解決するための行動計画を、フェーズ、具体的な行動内容、担当者、期待される効果に落とし込んで表形式で整理。
    - 会議の議事録音声などをソースにすれば、「誰が」「いつまでに」「何を」するべきかを大づかみで即座に理解できる。
- 14:13 <https://youtu.be/N77DlHnbD7Y?t=14m13s> シーン6：顧客相談後の課題整理と部署間連携
    - 商談や顧客とのやり取りの録音音声から、現在の状況に対する課題と改善の方向性を一覧化。
    - さらに、今後発生するタスクを「コンサルタント部」「調査部」「分析部」などの各部署へ自動で振り分け、優先度と共に整理して社内連携を円滑にする。
- 15:51 <https://youtu.be/N77DlHnbD7Y?t=15m51s> シーン7：外部環境調査（複雑なデータの早分かり）
    - 自治体の人口動態や世帯構成、産業就業者数といった、ページ数の多い複雑な外部環境調査データを一瞬で構造化。
    - スプレッドシートに展開することで、大量の定量的データを俯瞰し、一目で全体像を把握できるようにする。
- 17:38 <https://youtu.be/N77DlHnbD7Y?t=17m38s> 総括とエンディング
    - 今回紹介した7つのシーンは一例であり、自社の業務に合わせた無限の活用アイデアが考えられる。
    - NotebookLMのデータテーブル機能をビジネスの効率化にぜひ役立ててほしい。
